Para instalar esta herramienta debemos de clonarla de git 

	https://github.com/Mebus/cupp.git

Para iniciar la herramienta utilice

	Python3 cupp.py

Para crear el diccionario ejecutamos los siguiente

	Python3 cupp.py -i
